电商商品图片提取器使用文档

从 Amazon、AliExpress、SHEIN 商品页和搜索结果页中预览、勾选、批量下载商品图片。

一、插件简介

电商商品图片提取器是一款 Chrome 浏览器插件，用于从电商平台页面中识别商品图片，并在弹窗中预览、勾选、批量下载。

Amazon
AliExpress / 速卖通
SHEIN

支持商品详情页和搜索结果页。详情页提取商品主图、SKU 图、颜色/款式变体图；搜索结果页提取当前已加载商品的封面图。

二、安装方式

  1. 打开 Chrome 浏览器。
  2. 在地址栏输入 chrome://extensions/ 并回车。
  3. 打开右上角的“开发者模式”。
  4. 点击“加载已解压的扩展程序”。
  5. 选择本插件所在文件夹。
  6. 安装成功后，浏览器右上角会出现插件图标。

如果更新了插件文件，需要回到扩展管理页，点击该插件卡片上的“重新加载”按钮。

三、商品详情页使用方法

  1. 打开支持平台的商品详情页。
  2. 等页面商品图片正常加载出来。
  3. 点击浏览器右上角的插件图标。
  4. 在弹窗中查看已识别的图片分组。
  5. 勾选需要下载的图片。
  6. 点击“下载选中图片”。
  • Amazon 会优先识别 SKU/变体图片。
  • AliExpress 会优先识别颜色、款式等带图片的 SKU。
  • SHEIN 如果 SKU 选项本身没有图片，会自动退回采集商品主图图库。
  • 重复图片会尽量自动去重，只保留一张。

四、搜索结果页使用方法

  1. 打开支持平台的搜索结果页。
  2. 等当前页面商品列表加载出来。
  3. 点击浏览器右上角的插件图标。
  4. 在弹窗中查看当前已识别的商品封面图。
  5. 勾选需要下载的商品图片。
  6. 点击“下载选中图片”。

搜索结果页只采集当前页面已经加载出来的商品，不自动翻页，也不会自动打开商品详情页抓 SKU 图。

五、下载目录

图片会下载到浏览器默认下载目录下，并按平台自动建文件夹。同一平台的详情页图片和搜索页图片都会放在同一个平台文件夹里，来源信息会写进文件名。

六、常见问题

插件提示当前页面不支持

请确认当前页面是 Amazon、AliExpress 或 SHEIN 的商品详情页/搜索结果页。如果刚刚安装或更新插件，请刷新当前网页后再试。

图片识别数量太少

请等待页面图片加载完成，或向下滚动页面加载更多图片，然后点击插件弹窗右上角的刷新按钮重新读取。

SHEIN 详情页没有 SKU 图片

有些 SHEIN 商品只有尺码选项，没有颜色/款式图片。遇到这种情况，插件会采集商品主图图库，而不是尺码按钮。

修改插件后没有生效

请先在 chrome://extensions/ 中点击插件的“重新加载”，再刷新正在使用的电商网页。

七、注意事项

  • 插件只下载当前浏览器页面中可识别到的图片。
  • 插件不绕过登录、权限或平台限制。
  • 插件不自动批量翻页。
  • 插件不处理评论页、店铺页、视频页。
  • 下载图片前，请确认用途符合平台规则和图片版权要求。
