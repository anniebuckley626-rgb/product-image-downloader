# 隐私政策

最后更新：2026-06-20

亚马逊商品图片下载助手是一款用于识别、预览、勾选并批量下载商品图片的 Chrome 扩展。

## 数据收集

本扩展不会收集、存储、出售或传输用户个人数据。

本扩展不会收集以下信息：

- 个人身份信息
- 健康信息
- 财务和付款信息
- 身份验证信息
- 个人通讯内容
- 位置信息
- 浏览历史记录
- 用户活动数据

## 页面访问

本扩展仅在用户主动使用扩展时访问当前支持的网站页面，包括 Amazon、AliExpress 和 SHEIN。

扩展会读取当前页面中已经展示的商品图片、商品标题和商品标识信息，用于在本地生成预览列表，并帮助用户下载其主动选择的图片。

这些信息仅用于扩展的核心功能，不会上传到任何服务器。

## 下载权限

本扩展使用 Chrome 的 downloads 权限，将用户主动选择的商品图片保存到本地下载目录。

## 远程代码

本扩展不使用远程代码。所有 JavaScript、HTML 和 CSS 均包含在扩展安装包中，不会从远程服务器下载、执行或动态加载任何代码。

## 第三方服务

本扩展不会将用户数据发送给第三方服务。
